## 📋 API 接口清单

### 认证相关（2 个）
- POST /api/auth/wxlogin - 微信登录
- GET /api/user/info - 获取用户信息

### 首页相关（1 个）
- GET /api/home - 获取首页数据

### 商品相关（5 个）
- GET /api/goods/list - 商品列表
- GET /api/goods/search - 搜索商品
- GET /api/goods/:id - 商品详情
- GET /api/categories - 分类列表
- GET /api/styles - 风格列表

### 购物车相关（4 个）
- GET /api/cart - 获取购物车
- POST /api/cart - 添加到购物车
- PUT /api/cart/:cartId - 更新购物车
- DELETE /api/cart/:cartId - 删除购物车商品

### 收藏相关（3 个）
- GET /api/favorites - 收藏列表
- POST /api/favorites - 添加收藏
- DELETE /api/favorites/:goodsId - 取消收藏

### 订单相关（5 个）
- POST /api/orders - 创建订单
- GET /api/orders - 订单列表
- GET /api/orders/:orderId - 订单详情
- POST /api/orders/:orderId/cancel - 取消订单
- POST /api/orders/:orderId/confirm - 确认收货

### 地址相关（4 个）
- GET /api/addresses - 地址列表
- POST /api/addresses - 添加地址
- PUT /api/addresses/:addressId - 更新地址
- DELETE /api/addresses/:addressId - 删除地址

### 优惠券相关（1 个）
- GET /api/coupons - 优惠券列表

### 砍价相关（4 个）
- GET /api/bargain/goods - 砍价商品列表
- POST /api/bargain/start - 发起砍价
- GET /api/bargain/my - 我的砍价
- POST /api/bargain/:bargainId/help - 帮砍价

### 套餐相关（2 个）
- GET /api/packages - 套餐列表
- GET /api/packages/:packageId - 套餐详情

**总计：31 个 API 接口**
